ETIQUETAS DE TABLA EN HTML:

Subtag Description Example
<table> Creates a table <table><tr><td>Cell 1</td><td>Cell 2</td></tr></table>
<tr> Creates a table row <tr><td>Cell 1</td><td>Cell 2</td></tr>
<th> Creates a table header cell <th>Column 1</th><th>Column 2</th>
<td> Creates a table data cell <td>Cell 1</td><td>Cell 2</td>
<thead> Groups the header content in a table <thead><tr><th>Column 1</th><th>Column 2</th></tr></thead>
<tbody> Groups the body content in a table <tbody><tr><td>Cell 1</td><td>Cell 2</td></tr></tbody>
<tfoot> Groups the footer content in a table <tfoot><tr><td>Total</td><td>$100.00</td></tr></tfoot>
<caption> Adds a caption to a table <caption>Employee Salaries</caption>
<colgroup> Tag is useful for applying styles to entire columns, instead of repeating the styles for each cell, for each row. <colgroup>*col span="2" style="background-color:red"></colgroup>
<col> Specifies properties for a column in a table <col>col span="2" style="background-color:red"></col>

LOGICAMENTE ESTAS TABLAS PUEDEN SER EDITADAS CON STYLES

  1. <style> - defines styles for a table using CSS
  2. <div> - groups elements for styling purposes
  3. <span> - applies styles to specific text or elements within a table
  4. <a> - adds links to content within a table
